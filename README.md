A friendly Swedish-speaking hammerhead shark chatbot that helps with elementary school subjects!

## Features
- Helps with math, Swedish, science, social studies, and English
- Can teach Hebrew
- Over 200 sea-themed gum flavors in his collection
- Friendly personality with constant sea puns
- Rain animation effects

## Setup

1. Install dependencies:
\`\`\`bash
npm install
\`\`\`

2. Copy `.env.example` to `.env` and add your Supabase credentials

3. Deploy the Supabase function (see supabase/functions/chat-with-hamsa/)

4. Run locally:
\`\`\`bash
npm run dev
\`\`\`

5. Build for production:
\`\`\`bash
npm run build
\`\`\`

## Deployment

Deploy to Netlify or Vercel by connecting your GitHub repository.

Don't forget to add your environment variables in the deployment settings!
